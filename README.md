### Example Slurm Command on RHPC

The simulation study was run on the RHPC cluster. 

Below is the an example of shell commands to submit 3 (one for each *n* in {100, 1000, 10000}) array jobs to Slurm. The script uses 1 cpu per task. The default array specification in `sbatch_run_sim.sh` is 1-500 (for nsim=500 simulations) with a limit of a maximum of 100 concurrent tasks per job. As a task in an array, each simulation *i* job has a unique jobid of the format `${SLURM_JOB_ID}_${SLURM_ARRAY_TASK_ID}`; the `${SLURM_JOB_ID}` value is shared with all other simulations for sample size *n*.

```bash
# project directory
pdir=full/path/to/bios731_hw4_parrish

# script-make sure it is set to executable!
script=${pdir}/source/sbatch_run_sim.sh

# submit array job for each n
for n in 100 1000 10000; do
	# submission command
	submission_command='sbatch --job-name=n'${n}'_sim --export=ALL,n='${n}' --chdir='${pdir}'/logs --output='${pdir}'/logs/%x_%A_%a.out --error='${pdir}'/logs/%x_%A_%a.err '${script}

	# print full command
	echo ${submission_command}

	# submit command and get jobid
	jobid=`${submission_command} | cut -d" " -f4`

	echo ${jobid}" submitted"

done
```

A file for each sample size *n* and simulation *i* is saved as `./sim/sim_data/sim_n${n}_${i}.Rds` and can be copied from the cluster to a local folder using `scp`. An example command is shown below:

```bash 
scp user@cluster:/full/path/to/bios731_hw4_parrish/sim/sim_data/sim_n${n}_${i}.Rds /local/full/path/to/bios731_hw4_parrish/sim/sim_data/
```

Once results are finished, concatenate the estimates results using:
```bash
# loop over sample sizes
for n in 100 1000 10000; do

	# loop through simulations
	for i in {1..500}; do
		cat ./sim/sim_out/sim_n${n}_i${i}.txt >> ./sim/sim_n${n}.txt
	done

done
```

The concatenated estimate results from my simulation are found in the `./sim/sim_n${n}.txt` files available in this repo.
